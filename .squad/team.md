## Project Context

- **Owner:** anesterov
- **Project:** ARO Modernization Demo — "From VMware VMs to Containers on ARO"
- **Goal:** Demo (slides + recorded video) showing modernization from VMs on ARO Virt into containers on ARO, using GitHub Copilot for modernization assistance and OpenShift Service Mesh + Kiali to visualize the journey. Keep it simple and impactful.
- **Universe:** The Expanse
- **Created:** 2026-04-02
- **Stack:** ARO (OpenShift 4.x), ARO Virt (KubeVirt), OSSM (Istio-based), Kiali, ArgoCD, Tekton / GitHub Actions, RHEL VMs, GitHub Copilot
- **App baseline:** [Azure-Samples/modernize-monolith-workshop](https://github.com/Azure-Samples/modernize-monolith-workshop) — stepped branches used as baseline rather than live coding
- **Scope:** We do NOT need to modernize the full app. Show the process: ease of tooling, GHCP-assisted steps, OSSM blue-green, Kiali visualization.
- **Story arc:** Why migrate from VMware → "two worlds, one platform" (ARO Virt + containers) → modernizing with GHCP
